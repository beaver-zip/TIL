## Line break
- \<br> : a single line break
- \<p> : group a paragraph
- \<p style="margin-top:--px> : spacing between paragraph

\<p> is better choice than \<br> because it makes the boundary of paragraph clearly and can be changed freely by CSS.


## Why HTML matters
#### Example 1: Suppose we write a post on a blog.
- A : \<strong>\<span style="font-size:22px;">coding\</span>\</strong>
- B : \<h3>coding\</h3>
- Search Engine will show B much before than A.

#### Example 2: Suppose that a visually impaired person who use the screen reader visit our website.
If we create a web page using only images without using html tags, it will be darkness to them.
Conversely, if we use HTML tags sementically well, we can really help someone without even realizing it.

As such, html is important not only in terms of business but also in terms of humanism.
